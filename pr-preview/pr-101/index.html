High Performance Computing

Seamless access to advanced computing resources, consultation services and expertise for research.

High Speed Research Network

The High Speed Research Network (HSRN) is NYU's high-throughput, low-latency computer network dedicated to supporting data-intensive sciences.

GenAI

The Pythia Platform is a suite of tools for researchers to harness generative AI including a secure access to a chat portal, API access to LLMs and an on-prem vector database.

Cloud Computing

We facilitate access to Google Cloud Platform and also host an on-prem OpenShift cluster called Research Technology Cloud.

SRDE

The Secure Research Data Environment (SRDE) is a centralized secure computing platform designed to support research projects that require storage and computational resources specifically for sensitive data.

Find the Right Storage for Your Research

Not sure which storage solution is right for your research data? The Storage Finder tool helps you identify the best storage options based on your specific needs. Answer a few simple questions about your data requirements, and well help you make the right choice.
